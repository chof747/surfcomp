NAME

convert - surface file format conversion

SYNOPSIS

surfaceconvert [ -h -d ] [ -i <INFILE>] [ -o <OUTFILE> ] <INPUT_FORMAT> <OUTPUT_FORMAT>

DESCRIPTION

surfaceconvert is the file format conversion tool of the libsurface library. It automatically converts between all surface file formats available in the libsurface library. Which file formats are actually available, depends on the installed library version. A call of surconvert -h will provide you with a list of available file formats.
The tool expects a surface definition in the INPUT_FORMAT from either stdin or the specified INFILE and writes a surface in the OUTPUT_FORMAT to either stdout or the specified OUTFILE. It is furthermore possible to scan the input surface for duplicated surface points (i.e. points that have the same 3D coordinates) (option

OPTIONS

-h
print a short help message
-d
turn on the doublett checking and remove all duplicated points.
-i
the name of the input file. (if not specified stdin is used as input)
-o
the name of the output file. (if not specified stdout is used as input)
The options have to be followed by the name of the input file format and the name of the output file format.

DIAGNOSTICS

The surfaceconvert program uses the following exit codes:
0
everything went just fine
1
the program detected problems during the file input. These can either be caused by a misspelled filename or by the fact that the input is not formatted in the input format.
2
error during the output generation. This is most probably caused by a bad filename.
