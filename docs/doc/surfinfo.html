NAME

convert - surface file format conversion

SYNOPSIS

surfinfo [ -h -d ] [ -i <INFILE>] [ -f <FILEFORMAT> ] <INFORMATION>

DESCRIPTION

surfinfo provides the user with various information about the given molecular surface. The information is printed as a plain value on the std output so that it can be used by scripts and other program.

OPTIONS

-h
print a short help message
-d
turn on the doublett checking and remove all duplicated points.
-i
the name of the input file. (if not specified stdin is used as input)
-f
the file format of the input
The options have to be followed by one or more information tags from the following list:
points
the number of points on the surface
triangles
the number of triangles on the surface
edges
the number of edges on the surface
area
the surface area in square Angs.

DIAGNOSTICS

The surfinfo program uses the following exit codes:
0
everything went just fine
1
wrong command line parameter specification
2
the program detected problems during the file input. These can either be caused by a misspelled filename or by the fact that the input is not formatted in the input format.
3
an unknown input tag was specified.
