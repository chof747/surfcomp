This list of tutorials should guide you through the possible applications of the surfcomp program suite. If Sybyl® is available on your site you should start with the Sybyl tutorials. If you do not want to or if you cannot prepare your experiments with Sybyl follow the steps described in the MSMS tutorials.

In order to run all this tutorials you should have the surfcomp suite installed on your system and either the MSMS or Sybyl packages must be setup properly. In addition you should have a read/write access to a MySQL database server somewhere on your local system or in your network. How these packages are installed and set up can be learned from the surfcomp installation instructions or from the setup descriptions of the external software packages.

surfcomp with Sybyl

tutorial objectives
Compare proteins This lesson explains how protein surfaces can be prepared for an active-site-surface similarity search. It guides you step-by-step from a PDB file to the generation of the surfaces and data files to the comparison of a pair of active sites by surfcomp.
Compare ligands In this session you learn how to prepare and compare a set of small molecule/ligand surfaces. The structure of this lesson is similar to the Compare proteins tutorial.

surfcomp with MSMS

tutorial objectives
Compare surfaces by MSMS In this session you learn how to prepare and compare a set of small molecule/ligand surfaces with the MSMS and propgenerator programs. It is furthermore described how to launch the calculation.
