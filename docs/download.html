The surfcomp program files are available as source code, SRC RPM packages or binary RPM packages for different distributions. Look at the following table to choose the appropriate package. The suite requires some additional packages most of which are included in standard Linux distributions. You can find a list of these prerequisites below.

Additional requirements:

package description/dependency
MySQL The surfcomp programs store the surface data and fundamental parameters for the experiments within a local or remote MySQL database.
needed for all modules except libsurface!
Xerces-C A parser library for XML files. Because of their structure the experimental results are stored as XML files.
needed for the surfcomp and surfcomp-monitor programs
Geomview An interactive 3D viewer which can display arbitrary 3D scenes. This program is used by the viewer to bring the surface similarities on the screen.
needed for the surfcomp-monitor program
